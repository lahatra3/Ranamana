# Ranamana

- Ranamana est un chat bot (web) qui utilise l'api OPENAI.
- Il est créé avec la technologie [angular](https://angular.io)

- [ ] Descriptions:

``` Bash
    $ ~ cd namana
    $ ~ npm install
    $ ~ ng serve
```

- [ ] Remarques: 
  - La version d'angular utilisée de `14.12.0`.
